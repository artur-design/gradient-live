  <h1>Gradient live</h1>

Gradient live - это приложение-живые обои на андроид для дизайна и персонализации и представляет собой плавно меняющийся градиент. Оно имеет различные настройки для создания нужных цветов, скорости, плавности. Для открытия настроек достаточно дважды нажать на обои. Имеются следующие настройки:
  • Частота кадров
  • Количество кадров
  • Коэффициент длины градиента
  • Направление движения
  • Правила генерации цвета
  
  Для экономии батареи есть:
  • Черные обои
  • Остановка, если обои перекрываются другими приложениями

Также есть примеры для правил генерации цвета.
